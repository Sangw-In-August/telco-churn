프로젝트 개요

이동통신사의 고객 인적사항, 사용한 서비스 유형, 이용기간 등 다양한 데이터를 활용하여 고객의 서비스 이탈 가능성을 예측해 보고 이탈 요인을 분석해보았습니다.

데이터 로드

  • 전체 데이터 수: 7,043
  • 타겟 변수: Churn (서비스 이탈 여부)
  • 주요 변수: 성별, 자녀, 배우자 등 인적사항, 사용하고 있는 서비스의 유형, 부가 서비스 사용 여부, 월 요금 등
  • 데이터 출처: Kaggle

데이터 탐색 및 전처리

1) 데이터 타입, 유니크 밸류, 결측치 탐색

  • 타겟변수: Churn
  • 전처리 사항
    • TotalCharges 변수는 연속형 변수 임에도 불구하고 데이터 타입이 ‘Object’(문자열) 임을 발견 → 해당 변수에 포함된 일부 데이터가 공백(White space) 으로 입력되어 있음을 발견했고 이를 모두 결측치로 변환

2) 예측변수 분포

예측변수 중 연속형 변수의 분포 모형과 변수들 간의 관계를 나타내었습니다. 이를 통해 해당 데이터 셋이 여러 개의 독립적인 분포 모형(Multi-modal Distribution)을 가지고 있음을 예측

3) 연속형 변수의 상관관계

상관관계가 높은 변수들

TotalCharges ~ MonthlyCharges (0.65)
TotalCharges ~ tenure (0.83)

연속형 변수들 간에 상관관계를 탐색해 본 결과 TotalCharges 변수는 다른 변수들과 높은 상관관계로 인해 다중공선성을 일으킬 수 있기 때문에 제거

모델링 결과 탐색

1) 로지스틱 회귀 계수

  • 로지스틱 회귀계수의 절대값이 가장 큰 변수부터 내림차순으로 정렬하고 Logit으로 표현되는 회귀계수들을 해석이 용이한 오즈비로 변환
  • 로지스틱 회귀모형에 따르면 서비스 이탈 여부에 가장 크게 영향을 미치는 변수는 tenure (서비스 이용기간) 변수, Contract_Two year (계약 기간 2년 인지 여부) 등 임을 확인

2) Feature importances of XGBoost

  • XGBoost 모델을 이용한 변수의 중요도를 시각화하여 탐색해 본 결과 중요도가 높은 변수는 로지스틱 모형과 똑같지는 않지만 어느정도 비슷함을 확인

3) 중요변수 시각화

로지스틱 모델과 XGBoost 모델에서 공통으로 중요도가 높은 3개의 변수들을 추출하여 이 변수들이 서비스 이탈과 어떤 관계를 가지는 지를 알아보기 위해 시각화하여 탐색해 본 결과:

  • tenure (서비스 이용기간) 이 길수록 서비스 이탈율이 낮아진다.
  • 1, 2 년 단위 계약 보다는 월 단위 계약의 이용자가 이탈율이 더 높다.
  • 인터넷 서비스 중 Fiber optic (광섬유) 옵션을 이용하는 이용자의 이탈율이 높다.

클러스터링 분석

1) T-SNE 분석

  • T-SNE 분석을 통해 다차원 데이터 세트를 2차원으로 압축하여 이를 KDE 플롯을 활하여 시각화 했습니다. 시각화 결과 해당 데이터 세트는 다양한 분포가 독립적으로 구성되어 있음(Multi-modal Distribution)을 다시 한 번 확인
  • 이를 통해 해당 데이터셋은 각각의 분포를 가지는 독립적인 집단별로 서로 다른 서비스 이탈 패턴을 가질 수 있음을 가정

2) K-means Clustering

  • K-means 클러스터링을 수행하여 데이터 세트를 가장 유의미 하다고 판단되는 4개의 서로 다른 군집(G1, G2, G3, G4) 으로 분할
    • Elbow plot으로 최적 군집 수를 확인

3) 군집별 특성 히트맵

  • 고객 이탈율(Churn) 은 G1 > G2 > G3 > G4 순으로 높음.
  • 고객 이탈율이 가장 높은 G1 그룹은 서비스 이용기간이 상대적으로 짧으며, 연 단위가 아닌 월 단위로 서비스 계약을 맺는 특성을 가짐.
  • G2 그룹은 MonthlyCharges (월 요금)은 낮지만, MultipleLines, PhoneService (전화 서비스) 이용률이 낮은 특성을 가지고 있음. 이탈율은 상대적으로 높은 편.
  • G3 그룹은 월 요금은 다른 그룹들에 비해 가장 높지만 다양한 서비스를 복합적으로 이용하고 있는 특성이 있어 서비스 이탈율이 상대적으로 낮음.
  • G4 그룹은 서비스 이탈율이 가장 낮은 그룹으로 전화 서비스 이외에 인터넷, TV, 부가서비스 등 다른 서비스는 거의 이용하지 않는 특성이 있으며, 월 요금 또한 가장 저렴한 그룹임.
